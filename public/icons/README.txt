PWA Icons Required:

Place these PNG files in this directory:
- icon-192.png  (192x192 pixels)
- icon-512.png  (512x512 pixels)

Design recommendations:
- Dark background (#0a0a0a or #111111)
- Neon green or cyan lightning bolt / F logo / dumbbell icon
- Rounded corners (for Android adaptive icons use maskable purpose)
- Use tools like Figma, Canva, or https://realfavicongenerator.net

The icons are referenced in:
- vite.config.ts (PWA manifest)
- index.html (apple-touch-icon, favicon)
